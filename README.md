## Selected Projects

### Modern Energy Calculator

Desktop and web application designed to support nutrition students in the creation of meal plans and energy requirement calculations.

The original version was developed in Java Swing as part of my social service experience. It automates calculations related to basal energy expenditure, physical activity factors, macronutrient distribution and food equivalent systems. The project was later migrated to JavaFX and also adapted into a web application using Java Spring Boot.

**Main contributions:**

- Designed and implemented the main calculation workflow.
- Built the food equivalent system with editable portions and automatic nutrient updates.
- Integrated Excel-based food data recovery.
- Developed meal plan generation features.
- Added export functionality for generated meal plans.
- Improved the interface and adapted the system to a web environment.

**Technologies:** Java, Java Swing, JavaFX, Spring Boot, Maven, Excel processing, HTML, CSS, JavaScript.

---

### Sentiment Analysis Prototypes for Music Listening Behavior

Academic and thesis-oriented prototypes focused on analyzing song lyrics and user listening behavior through natural language processing.

The main idea is to support psychological interpretation by identifying emotional patterns, contradictions, sentiment tendencies and relevant linguistic signals in the music a user listens to. The system is not intended to replace professional judgment, but to provide structured insights that may help a psychologist observe possible emotional indicators.

**Main contributions:**

- Built multiple sentiment analysis prototypes.
- Worked with NLP preprocessing techniques such as tokenization, normalization and lemmatization.
- Explored rule-based analysis, lexicons, transformers and text classification strategies.
- Designed experimental approaches for interpreting lyrics beyond simple positive or negative labels.

**Technologies:** Python, Pandas, NLTK, Stanza, NLP, sentiment analysis, text processing.

---

### Continental Card Game

Web-based implementation of the Continental card game, focused on competitive gameplay, rule handling and responsive visual interaction.

The project involved organizing and improving an initial codebase where HTML, CSS and JavaScript were tightly mixed. I implemented complex card behavior, custom notifications, visual improvements and mobile-friendly interface adjustments.

**Main contributions:**

- Implemented joker behavior with flexible placement and exchange logic.
- Added Ace behavior with multiple possible values.
- Developed card ordering, construction jumps and validation logic.
- Designed custom card visuals and interface elements.
- Improved desktop and mobile experience.

**Technologies:** HTML, CSS, JavaScript, Node.js.

### Prototipos de Análisis de Sentimientos en Canciones

Prototipos académicos orientados a mi tesis, enfocados en analizar letras de canciones y comportamiento musical mediante procesamiento de lenguaje natural.

La idea principal es aportar información estructurada que pueda ayudar a interpretar posibles patrones emocionales en la música que escucha un usuario. El sistema no busca sustituir el criterio profesional de un psicólogo, sino ofrecer indicadores que puedan apoyar la observación y el análisis.

**Aportes principales:**

- Desarrollo de múltiples prototipos de análisis de sentimientos.
- Uso de técnicas de tokenización, normalización y lematización.
- Exploración de análisis basado en reglas, léxicos, clasificación textual y transformers.
- Diseño de enfoques experimentales para interpretar letras más allá de etiquetas simples como positivo o negativo.

**Tecnologías:** Python, Pandas, NLTK, Stanza, PLN, análisis de sentimientos.
